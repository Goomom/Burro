# Site Simples — Como usar

Arquivos criados:
- [index.html](index.html)
- [css/styles.css](css/styles.css)
- [js/script.js](js/script.js)

Como abrir:

- Método rápido: abra o arquivo `index.html` no seu navegador (duplo clique).

- Pelo PowerShell (na pasta `c:\Users\Guaxi\OneDrive\Desktop\html`):

```powershell
start .\index.html
```

- No VS Code: instale a extensão Live Server e clique em "Go Live" para ver alterações em tempo real.

O que cada arquivo faz:
- `index.html`: estrutura da página (conteúdo e links para CSS/JS).
- `css/styles.css`: estilos visuais (cores, espaçamento, botões).
- `js/script.js`: comportamento interativo (clique no botão, envio do formulário).

Próximos passos sugeridos:
- Aprender CSS para personalizar layout e responsividade.
- Aprender JavaScript para adicionar mais interações.
